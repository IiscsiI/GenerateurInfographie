❓ Aide

🚀 Demarrer rapidement

  • Cliquez 📚 Modeles pour partir d'un modele pre-rempli (cyberattaque, incendie, phishing, etc.)
  • Modifiez le titre, le sous-titre et les etapes selon votre besoin
  • Cliquez 📥 Export pour generer un PDF, PNG ou HTML autonome

✏️ Mise en forme du texte

Les boutons de la barre d'outils sous chaque zone de texte permettent de mettre en forme le contenu selectionne :

  • B = gras, I = italique, U = souligne, S = barre
  • ⬅ ⬆ ➡ = alignement du texte (gauche/centre/droite)
  • A = couleur du texte (ouvre le selecteur de couleur)
  • ↵ = saut de ligne, • = puce

🗄️ Case a cocher (listes)

Pour ajouter une case a cocher dans le texte d'une action (utile pour les checklists sur l'export HTML) :

<label><input type="checkbox"> Verifier les sauvegardes</label>

🎨 Apparence

  • Section Couleurs et themes : appliquer un theme pre-defini ou personnaliser chaque couleur
  • Toolbar Police : choisir une police globale pour le document
  • Toolbar A- / A+ : ajuster la taille globale de la police (50% a 200%)
  • Chaque etape peut avoir une couleur de categorie (bordure gauche rouge/bleu/orange/vert)

⌨️ Raccourcis clavier

  • Ctrl+S : sauvegarder
  • Ctrl+Z : annuler
  • Ctrl+Y ou Ctrl+Shift+Z : retablir

🔄 Reorganiser

  • Pour reorganiser les etapes : glisser-deposer via la poignee ⋮⋮ a gauche
  • Pour reorganiser les actions d'une etape : glisser-deposer via la poignee ⋮ a gauche de chaque action
  • Les numeros d'etapes se mettent a jour automatiquement

💾 Sauvegarde

  • Sauvegarde automatique toutes les 800 ms apres chaque modification
  • L'indicateur dans la barre d'outils affiche l'etat : Sauvegarde ✓ / en cours / non sauvegarde
  • Une derniere sauvegarde est tentee lors de la fermeture du navigateur

📥 Export

  • HTML autonome : fichier unique avec toutes les donnees, lisible dans n'importe quel navigateur
  • PDF : choisir format (A4/A3/A2) et orientation (portrait/paysage), 1 page garantie
  • PNG : image haute resolution (150 ou 300 DPI) pour impression ou insertion dans un document
