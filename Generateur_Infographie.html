💾 Gestion de projet

❓ Guide d'utilisation
🚀 Démarrage rapide ▼

Pour créer votre infographie :

  1. Personnalisez les couleurs selon votre charte graphique
  2. Modifiez le texte (titre, sous-titre, message d'urgence)
  3. Ajustez les étapes selon vos procédures internes
  4. Ajoutez votre logo pour personnaliser davantage
  5. Prévisualisez en temps réel vos modifications
  6. Exportez en HTML pour utilisation autonome
🎨 Personnalisation des couleurs ▼

Adaptez les couleurs à votre organisation :

  • En-tête : Couleur principale de votre organisation
  • Timeline : Couleur des étapes de progression
  • Arrière-plans : Personnalisez le fond de chaque section
  • Classifications : 4 niveaux de priorité personnalisables
  • Thèmes prédéfinis : Sélection rapide de palettes
📝 Gestion des étapes et actions ▼

Organisation des procédures :

  • Étapes : Grandes phases de la procédure (Immédiat, Alerte, etc.)
  • Actions : Tâches spécifiques dans chaque étape
  • Icônes : Plus de 40 émojis pour illustrer visuellement
  • Formatage : Gras, italique, soulignement, sauts de ligne
  • Priorités : 4 niveaux configurables (Critique, Important, Info, Succès)
🖼️ Logos et branding ▼

Intégration de votre identité visuelle :

  • Upload local : Glissez-déposez votre logo (PNG, JPG, SVG)
  • URL externe : Lien vers un logo hébergé en ligne
  • 6 positions : Haut/bas × gauche/centre/droite
  • Redimensionnement : Taille ajustable de 50 à 300px
  • Export intégré : Logo converti en base64 dans le HTML final
💾 Sauvegarde et export ▼

Gestion des projets et exports :

  • Sauvegarde automatique : Dans le navigateur (localStorage)
  • Export projet : Fichier JSON pour partage/backup
  • Import projet : Rechargement d'un projet sauvegardé
  • Export HTML : Page autonome avec tout intégré
  • Compatibilité : Fonctionne sur tous les navigateurs modernes
🎨 Personnalisation CSS avancée
ℹ️ Guide de personnalisation CSS ▼

Options de personnalisation CSS :

  • Préfixe de classe : Ajoute un préfixe unique à toutes les classes CSS pour éviter les conflits
  • Isolation CSS : Encapsule tous les styles dans un conteneur spécifique
  • CSS personnalisé : Ajoutez vos propres règles CSS qui seront incluses dans l'export
  • Reset CSS : Optionnel, réinitialise les styles de base

Exemple : Si vous utilisez le préfixe "cyb-", la classe ".container" devient ".cyb-container"
